О проекте

В 2018 году начата работа над созданием лексикографической базы диалектов даргинского языка Dargwa Dictionary Project. Это база, содержащая лексические данные различных диалектов даргинского языка и способная на дальнейшее расширение (как путем добавления новых диалектов, так и увеличения количества представленных в базе слов).

База предназначена как для лингвистов, так и для непрофессиональных пользователей – в основном для носителей различных даргинских идиомов. В сфере кавказоведения это первый масштабный проект такого типа: несмотря на существование печатных и цифровых словарей отдельных диалектов, задачи объединения и сопоставления сведений о современных даргинских языках на одной платформе до сих пор не ставилось.

С технической стороны проект представляет собой реляционную базу данных на платформе MySQL. Словарная статья в базе генерируется из массива базы данных по запросу пользователя. Сайт проекта разрабатывается с помощью микрофреймворка Flask на языке Python.
